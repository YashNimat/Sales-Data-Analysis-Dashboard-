# 📊 Sales Performance Dashboard (Power BI)

A comprehensive Sales Performance Dashboard built using **Microsoft Power BI** to analyze revenue, profit, sales trends, regional performance, and product category insights. The dashboard enables businesses to monitor key performance indicators (KPIs), identify top-performing products, and make data-driven decisions through interactive visualizations.

---

# 📌 Project Overview

The Sales Performance Dashboard is designed to provide an interactive overview of business sales performance. It consolidates sales data into meaningful visualizations that help users monitor revenue, profit, regional performance, product performance, and yearly sales trends.

The dashboard enables decision-makers to quickly identify profitable categories, best-selling products, and growth opportunities while improving overall business analysis.

# 📐 DAX Measures Used

Some commonly used DAX measures include:

```DAX
Total Revenue = SUM(Sales[Revenue])

Total Profit = SUM(Sales[Profit])

Profit Margin = DIVIDE([Total Profit],[Total Revenue])

Total Sales = SUM(Sales[Sales])
```
